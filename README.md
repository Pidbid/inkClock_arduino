# inkClock_arduino
Arduino写的墨水屏显示的时钟，所用MCU为ESP8266（NODE_MCUNODE_MCU）引脚定义在文件inkClock的第14行，屏幕采用微雪2.9inch 的墨水屏，外加一块18650电池和typec充电模块，和一个DHT11模块，实现温度年月日时的显示。右侧太阳图标是10S更换一侧位置，在凌晨0-8时会自动切换为月亮图标。所有模块都可以在TB购买，外壳采用3D打印制成。
